# NutriCart – Health-Focused E-Commerce Platform

NutriCart is a modern full-stack web application designed to promote mindful and healthy shopping. It allows users to explore, filter, and purchase health-oriented products while receiving personalized AI-driven nutrition recommendations. The project is built using the MERN stack with Next.js for a seamless and responsive user experience.

## Live Demo
Deployed on Vercel: https://nutri123.vercel.app/

## Overview
NutriCart combines e-commerce functionality with nutrition intelligence. Users can browse sustainable and nutritious products, manage their cart, and receive smart recommendations based on their dietary goals.

## Key Features
1. Dynamic Nutritional Filtering – Users can filter products based on nutritional values such as calories, protein, and fiber content.
2. AI-Powered Nutrition Recommender – Uses an AI model to suggest products aligned with the user's health goals such as weight loss, muscle gain, or vegan preferences.  
   It can analyze input data such as BMI or calorie intake and provide intelligent suggestions.  
   Example: “Based on your cart, you may want to include more fiber-rich items like Oats or Green Mix.”
3. JWT-Based Authentication – Secure login and signup system.
4. Redux-Powered Cart – Efficient and dynamic cart management.
5. Admin Dashboard – Enables product and user management with analytical insights.
6. Fast and Responsive UI – Built with React, TailwindCSS, and Next.js for optimal performance.

## Tech Stack
- Frontend: React.js, Next.js, TailwindCSS  
- Backend: Node.js, Express.js, MongoDB  
- Authentication: JWT (JSON Web Token)  
- State Management: Redux Toolkit  
- Hosting: Vercel  
- AI Integration: OpenAI API / Custom Model (for recommendation logic)
